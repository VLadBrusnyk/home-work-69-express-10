# ДЗ 69 - Операції з базами даних і документами в Mongo Shell

Цей проект реалізує завдання з Mongo Shell у вигляді RESTful API на базі Node.js, Express та Mongoose.

## Структура проекту

- `server.js` - Точка входу
- `config/db.js` - Підключення до MongoDB
- `models/Assignment.js` - Mongoose схема
- `controllers/assignmentController.js` - Логіка обробки запитів
- `routes/assignmentRoutes.js` - Маршрути API

## Встановлення та запуск

1. Встановіть залежності:

```bash
npm install
```

2. Запустіть сервер:

```bash
npm start
```

Сервер запуститься на http://localhost:3000

## API Endpoints

Всі маршрути починаються з `/api/assignments`.

| Метод  | URL              | Опис                                      | Завдання                                    |
| ------ | ---------------- | ----------------------------------------- | ------------------------------------------- |
| POST   | `/setup`         | Ініціалізує базу даних тестовими даними   | _Створіть колекцію... Додайте 5 документів_ |
| GET    | `/`              | Отримати всіх студентів                   | _Знайдіть всі документи_                    |
| GET    | `/?minScore=80`  | Отримати студентів з балом > 80           | _Знайдіть... де score > 80_                 |
| PUT    | `/bonus`         | Нарахувати +5 балів тим, у кого < 85      | _Оновіть один з документів..._              |
| DELETE | `/lowest`        | Видалити студента з найнижчим балом       | _Видаліть... найнижчий бал_                 |
| GET    | `/simple`        | Отримати тільки імена та бали             | _Проекція..._                               |
| GET    | `/stats`         | Отримати середній бал по предметах (> 75) | _Агрегаційні операції_                      |
| GET    | `/search?name=A` | Пошук за першою буквою імені              | _Робота з індексами_                        |
